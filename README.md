# NightBloom Frontend

This is the frontend for the NightBloom project. It is built using Next.js, MUI and TailwindCSS.  

First, run the development server:

```bash
npm install
npm run dev
# or
bun install
bun --bun run dev
```

To build the project for production, run:

```bash
npm run build
# or
bun install
bun --bun run build
```
